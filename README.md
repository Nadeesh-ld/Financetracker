## Features

✨ **Key Features:**

- 📊 **Dashboard Overview** - Get a quick overview of your financial status
- 💳 **Transaction Management** - Add, edit, and delete income and expense transactions
- 📈 **Financial Analytics** - Visual charts and graphs to track spending patterns
- 🏷️ **Category Tracking** - Organize transactions by categories
- 💾 **Data Persistence** - All your data is securely stored
- 🎨 **Responsive Design** - Works seamlessly on desktop and mobile devices
- 🔍 **Search & Filter** - Easily find specific transactions

## Installation

### Prerequisites

- A modern web browser (Chrome, Firefox, Safari, Edge)
- Git installed on your system

### Steps

1. **Clone the repository:**
   ```bash
   git clone https://github.com/Nadeesh-ld/Financetracker.git
   cd Financetracker
   ```

2. **Navigate to the frontend directory:**
   ```bash
   cd Financetrackerfrontend
   ```

3. **Open the application:**
   - Double-click `index.html` to open in your default browser, OR
   - Use a local server (recommended):
     ```bash
     # Using Python 3
     python -m http.server 8000
     
     # Using Python 2
     python -m SimpleHTTPServer 8000
     
     # Using Node.js (http-server)
     npx http-server
     ```
   - Then visit `http://localhost:8000` in your browser

## Usage

1. **Launch the Application** - Open the application in your web browser
2. **View Dashboard** - See your current financial overview
3. **Add Transactions** - Click the "Add Transaction" button to log income or expenses
4. **Categorize** - Assign each transaction to a category (e.g., Food, Transport, Salary)
5. **View Analytics** - Check charts and statistics to understand your spending habits
6. **Filter & Search** - Use the search functionality to find specific transactions
7. **Export Data** - Export your financial data for backup or analysis
